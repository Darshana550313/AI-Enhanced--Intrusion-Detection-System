About the Project

Objective

To develop a robust Intrusion Detection System using Machine Learning that can accurately distinguish between normal traffic and various attack types.

Methodology

  • Data: NSL-KDD Dataset (Primus11 Version)
  • Models: Decision Tree, Random Forest, XGBoost
  • Metrics: Accuracy, Precision, Recall, F1-Score

Attack Types

TypeDescription
NormalLegitimate network traffic
DoSDenial of Service attacks
